Dipendenze note:

- Le dipendenze andrebbero cercate seguendo l'ordine riportato di seguito.

1) Controllo della presenza di directories.
2) Mancano directory ?
	1) Quali directory mancano?
		1) Se mancano directory essenziali, invia messaggio di errore.
		2) Se mancano directory superficiali esegui programma di ripristino.
3) Controllo dei file all'interno della directories.
	1) Mancano dei file ?
		1) Se mancano dei file essenziali, invia messaggio di errore.
		2) Se mancano file superficiali, esegui programma di ripristino
4) Controllo dell'ultimo profilo inserito all'interno di last_user.txt.
	1) sono presenti utenti non guest ?
		1) Controllo della presenza di una file di conf per utente non guest:
			1) Se presente, considera questo per il conf del programma
			2) Se non presente
				1) Creazione file profilo utente con valori di default.
